## 定义

<a name="user"></a>
### User

|名称|说明|类型|
|---|---|---|
|**addr**  <br>*可选*|用户地址  <br>**样例** : `"string"`|string|
|**age**  <br>*可选*|用户年龄  <br>**样例** : `0`|integer (int32)|
|**userId**  <br>*可选*|用户 ID  <br>**样例** : `0`|integer (int32)|
|**userName**  <br>*可选*|用户名  <br>**样例** : `"string"`|string|
